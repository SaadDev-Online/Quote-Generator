# Quote‑Generator

A simple web application that generates random quotes for inspiration and sharing.

## Features

- Display a random quote and its author on button click  
- Responsive design suitable for mobile and desktop  
- Easy to extend with new quotes, categories or animations  

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Optional: external quote API or hard‑coded quote list
